# Project Pizzeria
Application for pizzeria written in JavaScript.
> Live demo [https://react-waiter-app.lemarczini21.repl.co/.](https://project-pizzeria.lemarczini21.repl.co)


## Technologies Used
- Vanilla JavaScript
- HTML
- CSS/SASS

## Features
- Make an order
- Table reservation
- WEB API
